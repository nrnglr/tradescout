# Logo Değişikliği

Yeni FGSTrade logosunu kullanmak için:

1. Gönderilen logo görselini bu klasöre `fgs-logo.png` olarak kaydedin
2. Aşağıdaki dosyalarda logo import'ları otomatik olarak güncellendi:
   - LandingPage.tsx
   - Login.tsx  
   - Register.tsx
   - Dashboard.tsx (navbar için)

## Logo Boyutları
- Landing Page: 55px yükseklik (responsive)
- Login/Register: 100-120px 
- Dashboard: 45px yükseklik

## Not
Şu an `image.png` yedeğ olarak kullanılıyor. `fgs-logo.png` eklendiğinde otomatik değişecek.
